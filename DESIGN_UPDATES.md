## 🎨 Color Palette

### Primary Colors:
- **Purple**: `#667eea` (Primary brand color)
- **Dark Purple**: `#764ba2` (Gradient accent)
- **White**: `#ffffff` (Backgrounds)
- **Off-White**: `#fafafa` (Page background)

### Text Colors:
- **Primary**: `#1a1a1a` (Headings, important text)
- **Secondary**: `rgba(0,0,0,0.6)` (Body text)
- **Muted**: `rgba(0,0,0,0.4)` (Subtle text)

### State Colors:
- **Success**: Green (stock available)
- **Warning**: Orange (low stock)
- **Error**: Red (out of stock)

---

## 📐 Typography

### Font Stack:
```css
-apple-system, BlinkMacSystemFont, 'Segoe UI', 'Roboto', 
'Helvetica Neue', Arial, sans-serif
```

### Font Weights:
- **Regular**: 400 (body text)
- **Medium**: 500 (buttons, labels)
- **Semibold**: 600 (subheadings)
- **Bold**: 700 (main headings)

### Letter Spacing:
- Large headings: `-2px` to `-1px` (tighter, more modern)
- Subheadings: `-0.5px`
- Body: default

---

## 🎭 UI Components

### Buttons:
- **Rounded corners**: `12px` border radius
- **Padding**: `px: 4, py: 1.5` (comfortable sizing)
- **Hover effects**: Scale + color change
- **Text transform**: none (natural capitalization)

### Cards:
- **Border radius**: `16px` (very rounded, modern)
- **Borders**: `1px solid divider`
- **Shadows**: Subtle, increase on hover
- **Hover animation**: `translateY(-8px)` lift effect

### Input Fields:
- **Border radius**: `12px`
- **Focus state**: 2px purple border
- **Background**: White with subtle border

---

## ✨ Animations & Transitions

### Hover Effects:
```css
transition: all 0.3s ease
```

### Card Hover:
- Lift up 8px
- Increase shadow
- Change border color

### Button Hover:
- Slight scale (1.02-1.05)
- Color shift
- Shadow increase

### Smooth Scroll:
- Enabled globally
- Smooth page navigation
